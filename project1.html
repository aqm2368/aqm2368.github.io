UI Design – Dash Cooldown System

← Back to Home

This project demonstrates my process for creating a responsive UI system in Unity. It includes cooldown animations, state transitions, and visual clarity designed for high-speed gameplay.

Goal: Create a visual language that reflects player momentum and timing.

Video Overview

Process

  • Prototyped layout in Google Slides for quick iteration
  • Designed assets in Krita/Photoshop at 4K resolution
  • Trimmed layers for efficiency before importing into Unity
  • Used Image.fillAmount and fillOrigin to animate recharge visuals
